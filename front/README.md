
# Esperanto




## Run Locally

Clone the project

```bash
  git clone https://github.com/PaulGub/esperanto.git
```

Go to the project directory

```bash
  cd esperanto
```



## Installation

Install dependencies with [pnpm](https://pnpm.io/fr/)

```bash
  npm install -g pnpm
```

For both front & back 
```bash
cd front / back 
pnpm i 
```
    
## Developement

Start the server

```bash
  pnpm dev
```
